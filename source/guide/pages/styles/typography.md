To use the alternate heading font, add the `.text-alternative` class.  For instance:

{% example html %}
<h2 class="text-alternate">Heading two</h2>
<p>Lorem Ipsum is simply dummy text of the printing and typesetting industry. Lorem Ipsum has been the industry's standard dummy text ever since the 1500s, when an unknown printer took a galley of type and scrambled it to make a type specimen book.</p>
{% endexample %}

The relative variables to change the defaults:

{% highlight sass %}
$font-heading: $font-whitman-display-compressed-bold !default;
$font-heading-alt: $font-benton-sans !default;
{% endhighlight %}
